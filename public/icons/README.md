## Icon Oluşturma:

1. Ana logo/icon'unuzu 512x512 boyutunda hazırlayın
2. Online araçlar kullanarak (realfavicongenerator.net gibi) tüm boyutları oluşturun
3. Dosyaları bu klasöre yerleştirin
